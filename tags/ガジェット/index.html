自作PC完成

つくることになったきっかけはいろいろあった気がするのですが、それはさておき、PCを組み立ててみました。

各パーツそれぞれ一つずつ調べたので、だいぶ時間はかかりましたが、これまで全く分かっていなかったマザーボードとかCPUとかについて、結構親近感を持てるくらいにはなったのは、もともとそこまでは期待してなかっただけにかなりの収穫でした。

CINEBENCH R15でベンチマークとってみたんですが、今まで動画編集に使ってたMacBook Pro（2014年モデルのHDD以外は一番スペック高いやつ）と比較しても、

MacBook Pro
OpenGL: 53.89 fps
CPU 583cb

自作PC
OpenGL: 121.25 fps（約2.25倍）
CPU 878cb（約1.5倍）

と、期待以上の結果が出てくれたので、これからコンテンツ作りがますます楽しみです。

というわけで、パーツ選びの後半はいろんな人が公開している組合せがめちゃくちゃ役に立ったので、各パーツとそれぞれの購入時の価格の一覧も一緒にまとめておきます。

HDD 12TB、メモリ64GB、グラフィックボードも最後迷って上のモデルにという、動画・グラフィック周りの処理を考えて作りました。
とりあえずまだ動画編集とかをやってないので、その部分はわからないのですが、HDDを12TBにしたのはかなりいいです。これまで外付けHDDを10台くらいがちゃがちゃやってたのが、これからこの1台で基本的に良いというのは、相当な気持ちの軽さを感じます。

ここまでしてコンテンツ作らないわけにはいなかないので、さっそくがつがつ作っていきたいと思います。

それにしても、使い勝手がよすぎるからなのか、自分でパーツ選んだりしたからなのか、組み立てたからなのか分かりませんが、かなり愛着と満足感と幸福感を感じます笑
全部これで済ませたい気持ちもあるのですが、開発周りはWindowsはつらいので、あくまでAdobe系の専用マシンという使い方をしていく予定です。

“組み立てた”というフレーズで思い出しましたが、パーツ選び（本読んだりも含めて）20時間くらい、組み立て〜インストールに12時間くらいかかったので、なんだかんだそこそこ時間かかってしまいました。これまでのWeb上で予算に合わせてちょちょっとカスタマイズして、ぽちってやってただけだったのと比べるとだいぶ労力使った感じはありますが、納得行くスペックのPCになったこと、パーツ関連の知識がついたこと、これからさらにグレードアップさせていけること、あと普通に買うより格段に安いことを考えると、自作PCを作ることはいろんな意味でプラスだと思います。
（7万円くらいでも組もうと思ったら、普通に組めるはずなので、そこから徐々にアップグレードしていくという方法もありだと思います）

今後強化するとすれば、もうメモリはこれ以上増やせないので、とりあえずビデオカードとSDD・HDDあたりかなあと思っていますが、少なくとも2~3カ月はこのままでいいかなと思っています。

携帯ぜんぶ解約

ついに、いま契約している携帯の回線を全て解約することにしました。
理由は2つだけで、一つがSIMフリーの便利さを知ってしまったため、そしてもう一つが料金が高すぎるためです。

SIMフリー便利

先月2台のiPhone 6sを購入したのですが、そのうち1台をSIMフリー端末にしたので、そのタイミングでSIMカードを久しぶりに購入してみました。
そのときに目に止まったのが、IIJmioのプリペードSIM。(https://s.iijmio.jp/prepaid/)
税込み4,000円ちょっとで1枚のSIMカードを購入して、決まった番号に電話をかけパスコードを入力すると使えるようになるという手軽さです。
3ヵ月の期限に達するか2GBを使いきるまで、そのまま使い続けることが出来ます。
電話もSMSも出来ませんが、僕は電話を使わないのでデータ通信だけで十分で、さらにテザリングも出来るので個人的なユースケースにはピッタリでした。

これがじわじわ身にしみる便利さで、携帯回線を毎月払うというしばりを一切感じる必要がなくなり、予備として2枚くらい持っておけば、上限に達したら交換すればいいというめちゃくちゃ身軽な状態になります。

余談
そして、この便利さがじわじわ身にしみてきたタイミングでNexus 5xが欲しくなりましたｗ
でもiPhone 6sを2台両手持ちとかしたばっかりだしなーちょっとさすがになーと迷いに迷って、結局Nexusは諦めよう！と思ったあたりで、
Now on Tapの動画をみて、やっぱOSでみたらiOSより断然Androidの方がいいよなーとか思っていて、迷いに迷って、結局Nexusは諦めよう！と思ったあたりで、
2ヵ月ぶりにAndroid使うかーと触ってみたら、断然やっぱりAndroidの方がいい！！！！と思ったんですが、迷いに迷って、結局Nexusは諦めよう！と思ったあたりで、
毎月の携帯の請求書がちょうどきて、やっぱ高いなーと思ったわけです。
そこで、ふと「もし携帯解約してSIMフリー端末だけにしたら長期的にみたら絶対コスト下がるんじゃないか？となると、Softbankの方のiPhone 6sは解約すべきじゃないか？そして、さらにソフマップで買い取り価格調べてみると結構高いということも判明！！
ということは、
iPhone 6sを解約して売りに出して、ランニングコストを下げるのが一番かしこいんじゃないか？
そしたらSIMフリーのNexus 5xを買ってもいいんじゃないか？？！（謎）
ということがこの携帯解約の経緯の裏側にあるので、結局Nexusが欲しかっただけの可能性は否定出来ません。

携帯料金高すぎ

メインの回線それぞれにかかっている料金はおおざっぱに見て

  • Softbank: 2万円弱
  • au: 1万円強
  • Y!mobile: 2万円弱

という感じで、月によっても結構変わるので、ざっくりみて4万3千円〜5万円ちょっとの間という感じです。（ちなみにこれプラスでWiMaxも持ってます）
これまではテザリングも使ってるし、これくらいでしょうがないかと言う気持ちもあったんですが、それにしても1Kの家賃分ぐらいずっとかかっているのは高すぎるし、上に書いたSIMの便利さもあるしということで、あまりメリットが無いのではないかと思えてきました。
ただ、iPhone 6sを当たり前の様に、分割で購入できたりというのはこの契約のプラスポイントですし、そのシステムを使わせてもらってるから、どんどん端末の分割代金が増えて料金が高くなってるのも事実なので、特に携帯会社が悪いというつもりは全く無いです。

今後

というわけで、

  • Softbank: 2万円弱
  • au: 1万円強
  • Y!mobile: 2万円弱
  • （使ってなかったiPad: 4千円弱）

を全て解約することにしました。
番号がほんとにないというのは厳しいので、1つの番号だけガラケーで月1,000円くらいで運用出来るようにしようと思ってます。

そんなわけで、
月々の料金が、
- 携帯利用料: - 5万円 - SIMカード: + 0.3〜0.5万円 - ガラケー: + 0.1万円

という計算により、大体少なく見ても毎月4万円は最低でもコストダウンできるということになりそうです。
スマホは生活の中心だし仕事でも重要なものだから、必要なんだと思いこんでいてあまり料金のことは考えてませんでしたが、思ったよりも削れそうなので、一通り見なおしてみてよかったです。

忘れてはいけないこと

携帯は分割で購入しているため、解約するタイミングでそれの残りを一括で支払う必要があります。 いくら支払うことになるのか分かりませんが、30万円くらいいくような気がします。
SIMフリー端末以外は全て売るつもりですが、それでどれくらいおぎなえるか…
